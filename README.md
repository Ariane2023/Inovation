# Innovation Class - Projeto de E-commerce (Front-end)
![image](https://github.com/user-attachments/assets/95ad5397-6a74-401b-920f-82f91c6b9644)


Este README fornece uma visão geral do projeto de front-end desenvolvido como parte da "Innovation Class". 
O projeto consiste em uma página inicial de um e-commerce com diversas seções, incluindo um header com busca,
categorias, banner principal, seção de lançamentos, informações adicionais, newsletter, rodapé e métodos de pagamento.
![image](https://github.com/user-attachments/assets/f918f587-299d-406c-afe9-444750af6b03)

## Tecnologias Utilizadas

* **HTML:** Estrutura da página e conteúdo.
* **CSS:** Estilização e layout da página (arquivo `index.css`).
* **JavaScript:** Adição de interatividade (arquivo `script.js`), especificamente para a funcionalidade de busca.
* **Fontes:** Google Fonts (`Nunito Sans` e `Nunito`).



## Como Executar o Projeto

Para visualizar este projeto, basta abrir o arquivo `index.html` em qualquer navegador web moderno.

## Melhorias e Próximos Passos

Este projeto pode ser aprimorado com as seguintes funcionalidades:

* Implementação de um dropdown funcional para os submenus de categorias.
* Criação de um carrossel dinâmico e interativo para a seção de lançamentos.
* Integração da funcionalidade de busca para realmente pesquisar produtos (requer back-end).
* Implementação de validação para o formulário de newsletter.
* Melhorias na responsividade para diferentes tamanhos de tela (design responsivo).
* Otimização de imagens e outros recursos para melhor desempenho.
* Adição de mais seções e conteúdo para uma página de e-commerce completa.
* Adição de banco de dados para captar os cadastros, utilizando linguagens como MySQl (requer back-end).
* Considerar a acessibilidade ao implementar novas funcionalidades, garantindo que todos os usuários possam interagir com o site.
  ![image](https://github.com/user-attachments/assets/daa149b6-b1fb-410b-a45d-2454839cd008)


## Observações

* O design e o conteúdo presentes neste projeto foram retirados do figma fornecido pelo projeto Innovation Class.
* Algumas funcionalidades, como a pesquisa real de produtos, requerem uma implementação de back-end que não está incluída neste projeto de front-end.
* A acessibilidade é uma consideração importante, e ao adicionar novas funcionalidades, deve-se garantir que o site seja utilizável por todos.
